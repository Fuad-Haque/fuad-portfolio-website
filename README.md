# fuadhaque.com

Full-stack portfolio — FastAPI backend + Next.js frontend.

## Structure
- `backend/` → FastAPI + PostgreSQL → Railway
- `frontend/` → Next.js 14 + TypeScript → Vercel

## Local Dev
```bash
cp .env.example .env
docker-compose up
```
- Frontend: http://localhost:3000  
- Backend: http://localhost:8000/docs
