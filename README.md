# Fuad Haque — Portfolio Website

Personal portfolio built with the "Obsidian Scholar" design language — merging dark academia aesthetics with a developer OS interface.

## Live Site
[fuad-haque.vercel.app](https://fuad-haque.vercel.app) ← update this after Vercel deploys

## Tech Stack
- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Features
- Draggable floating windows (OS-style interface)
- `Cmd+K` command palette with keyboard navigation
- Mouse-following spotlight effect
- SVG noise grain overlay
- Custom cursor with lagging ring
- Playfair Display × JetBrains Mono typography pairing
- Fully responsive (mobile stacked layout)

## Run Locally
```bash
git clone https://github.com/Fuad-Haque/fuad-portfolio-website.git
cd fuad-portfolio-website
npm install
npm run dev
```
